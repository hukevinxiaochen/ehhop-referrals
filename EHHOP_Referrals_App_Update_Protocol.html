Read about how to layout your change/update first:

A content change/update is a combination of substitutions, deletions, and additions. Please layout all desired changes as a list of instructions each phrased in one of the following ways:

  • Substitute \type[X] with \type[Y] in *Location*
  • Delete \type[X] from *Location*
  • Add \type[Y] to *Location*

Where  [X] is some information/content you want removed or deleted, and  [Y] is  information/content you want to take its place or to be added. *Location*  specifies where you want this instruction applied. Finally \type is some language  specifying what kind of object you are talking about substituting/deleting/adding.

How to specify *Location*

*Location* specifies where the change is to take place. It should be clear  with respect to: [1] on which page the substitution, deletion, or addition will occur,  and [2] where on that page it will go (in relation to everything else on the page).  Since the app is structured exactly like the folders/directories on a computer, specifying location is pretty easy.

How to specify [X] and [Y]

[X] will refer to what is to be removed. It should be clear from your language exactly what gets removed  It is important that [Y] be text/image representing exactly what is to take its place and be added.

How to specify \type

\type specifies the type of information/object [X] or [Y] belongs to. Use  a combination of plain english and the abbreviations below to help specify text you want to add/remove:

  • List Heading - \h
  • Checklist Item - >
  • List Item - -
  • Free Text - \p
  • Navigation Bar Button \nav
  • Link To: E-mail/Phone Number/Google Doc/Other URL - \go(url)
  • Image - \img

Putting it all together

Example: Substitute the entire ACT Checklist with "\pREMINDER: You can use your own account on EPIC to submit referrals requests>Submit  appointment request via \go(http://ehhop0.appspot.com/)EPIC >Confirm appointment when prompted >Send reminders to patient" in *ACT checklists all pages*

This change would look like:

Access to Care Team

REMINDER: You can use your own account on EPIC to make appointment requests with any department

You can make a spreadsheet  to help list out a whole bunch of content changes at one time.

Please don't hesitate to submit changes that do not necessarily fit in the framework presented above.  The instructions above are just some specific guidelines for basic content changes.
